
Overview
Class actions fit best where many people suffered similar harm from the same policy or product. Mass torts fit where injuries vary by person. Some issues are better as individual cases or regulatory complaints.

Legend
Class action strong fit
Class action possible with facts
Usually not class action. Better as individual, mass tort, or regulatory

Landlord and Housing
* Illegal evictions and lock changes without proper notice  Class action possible with facts. If a landlord or management company uses a uniform policy across many tenants
* Landlords demanding advanced rent payments beyond lease terms  Class action possible with facts
* Landlords withholding keys from tenants with valid leases  Class action possible with facts
* Unauthorized entry and touring of rental units with tenant belongings present  Class action possible with facts
* Failure to honor existing leases after property sale  Class action possible with facts
* Improper handling of security deposits  Class action strong fit. Common policy across many tenants
* Landlords giving unauthorized access to storage units  Class action possible with facts
* Missing or damaged belongings due to landlord negligence  Usually not class action. Injuries vary by person. Possible class only if a uniform event or policy
* Security equipment debt transferred to new homeowners  Class action possible with facts. If a seller or installer used a uniform contract practice
* Previous owners demanding payment for items left at property  Usually not class action unless a company runs this as a scheme
* Illegal lockouts and self help evictions  Class action possible with facts
* Refusal to provide keys after lock changes  Class action possible with facts
* Forced homelessness due to improper eviction procedures  Usually not class action. Damages vary. Could be part of a policy case
* Property managers mishandling maintenance issues  Class action possible with facts only if a habitability policy is uniformly breached
* Landlords changing locks and moving tenant belongings without permission  Class action possible with facts
* Failure to maintain habitable living conditions  Class action strong fit if policy or systemic neglect across many units
* Retaliatory actions against tenants who complain  Class action possible with facts but may vary by person

Employment and Workplace
* Employers withholding final paychecks to force contract signatures  Class action strong fit for wage and hour violations
* Non compete agreements demanded after employment ends  Class action strong fit for unlawful non compete practices
* Workplace requiring employee spouses to follow conduct codes  Class action possible with facts
* Tuition reimbursement demands after wrongful termination  Class action possible with facts
* HR failing to add dependents to insurance within enrollment windows  Usually not class action. Facts vary
* Supervisors failing to forward required documents to HR in timely manner  Usually not class action
* Mandatory insurance coverage gaps affecting newborns  Usually not class action
* Employers forcing employees to sign documents under duress  Class action possible with facts
* Workplace discrimination and harassment  Often individual or pattern and practice. A class can exist if a policy causes common harm
* Wage theft and unpaid overtime  Class action strong fit
* Wrongful termination without cause  Usually not class action
* Forced arbitration clauses in employment contracts  Class action strong fit for declaratory or injunctive claims. Mass arbitration strategy may apply
* Retaliation for reporting safety violations  Often individual. Class only if uniform retaliation policy
* Failure to provide required breaks  Class action strong fit
* Misclassification of employees as independent contractors  Class action strong fit

* Subscription services that make cancellation difficult  Class action strong fit
* Hidden fees and charges not disclosed upfront  Class action strong fit
* Bait and switch advertising tactics  Class action strong fit
* Unauthorized auto renewal of subscriptions  Class action strong fit

Financial and Banking

* Hospital billing errors and surprise medical bills  Class action possible with facts for uniform billing practice
* Ambulance companies charging for unnecessary transport  Class action possible with facts
* Police and paramedics forcing unnecessary hospital visits with billing  Usually not class action
* Credit card companies charging unauthorized fees  Class action strong fit
* Banks mishandling account access and security  Class action possible with facts
* Settlement funds stolen by family members or guardians  Individual probate or criminal issues
* Hidden settlement from minor beneficiary by parent  Individual probate issue
* Life insurance beneficiary disputes despite clear documentation  Usually individual
* Financial institutions refusing to provide account access  Class action possible with facts
* Predatory lending practices and excessive interest rates  Class action strong fit
* Improper handling of deceased persons accounts  Class action possible with facts

Family Law and Custody
* Foster care system failures leading to child abuse  Complex. Often civil rights class actions for injunctive relief
* Inadequate background checks for foster or adoptive parents  Class action possible for injunctive relief
* CPS removing children without proper investigation  Civil rights pattern and practice class possible
* Family members hiding inheritance from rightful heirs  Individual probate disputes
* Step parents attempting to claim deceased spouse property  Individual probate
* Custody disputes involving abuse allegations  Individual
* Failure to protect children from known abusers  Civil rights pattern and practice possible
* Guardians misappropriating funds meant for minors  Individual unless a system failure

Healthcare and Medical
* Medical debt from forced unnecessary treatments  Class action possible with facts on uniform billing practice

* Insurance not covering medically necessary procedures  Class action possible with policy driven denials
* Pharmacy errors and medication mix ups  Usually individual. Class possible for labeling or dispensing policy
* Hospital emergency department excessive wait times  Class action unlikely for damages. Possible injunctive claims
* Medical billing fraud and upcoding  Class action strong fit if systemic
* Denial of coverage for pre existing conditions  Class action strong fit if unlawful post law changes
* Improper handling of patient medical records  Class action strong fit under privacy laws
* Healthcare providers refusing to accept valid insurance  Class action possible with facts

Property and Real Estate

* Title companies failing to identify liens  Class action possible with systemic failures
* Real estate agents breaching fiduciary duties  Usually individual unless a policy
* Home warranties not covering claimed items  Class action strong fit if uniform denial scheme
* HOA overreach and excessive fines  Class action possible with facts
* Property tax assessment errors  Often administrative appeals. Class possible for uniform unlawful method
* Eminent domain abuse  Often individual or public law litigation

Technology and Privacy
* Data breaches and inadequate security measures  Class action strong fit
* Unauthorized use of personal information  Class action strong fit
* Companies refusing to delete personal data upon request  Class action strong fit for injunctive and statutory damages depending on law
* Facial recognition technology used without consent  Class action strong fit under biometric laws
* Social media platforms censoring users arbitrarily  Complex First Amendment concerns. Private platform policies. Class viability uncertain
* Smart device privacy violations  Class action strong fit
* Online platforms failing to protect minors  Class action strong fit and regulatory action
