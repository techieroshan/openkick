# Demo Script

## Investor Journey

### 1. Discover Offerings
- Navigate to `/invest`
- Browse available offerings
- Filter by status (open/funded/closed)
- Click on an offering to view details

### 2. View Offering Details
- See case overview, terms, risk disclosures
- Review funding progress and timeline
- Check minimum investment requirements

### 3. Complete Investor Profile
- Navigate to `/dashboard/investor`
- Create investor profile (name, address, DOB)
- Complete KYC upload and verification
- If required, complete accreditation verification

### 4. Invest
- Return to offering detail page
- Enter investment amount (must meet minimum)
- Acknowledge risk disclosure modal
- Confirm investment
- View investment in portfolio

### 5. View Portfolio
- Navigate to `/dashboard/investor`
- See all investments with status
- View distributions when available
- Download statements

## Issuer Journey

### 1. Create Campaign
- Navigate to `/invest/start`
- Submit case information
- Upload required documents
- Configure offering terms
- Submit for compliance review

### 2. Manage Campaign
- Navigate to `/dashboard/issuer`
- View campaign status and progress
- See investor list
- Post updates (requires approval)
- Manage media assets

## Compliance Officer Journey

### 1. Review Offerings
- Navigate to `/compliance`
- Review pending offerings
- Check documents and disclosures
- Approve or request changes

### 2. Review Communications
- Review issuer updates
- Approve or request changes
- Ensure compliance with regulations
